API Documentation

Complete reference for the Bank Management API. All endpoints require a Bearer JWT token unless marked Public. Admin-only endpoints are marked Admin.

Core Capabilities

Account Management

Multiple account types with real-time balances, statements, and joint account support.

  • Savings, Checking & Joint Accounts
  • Real-time Balance Tracking
  • Account Statements (PDF/JSON)

Transaction Processing

Secure transfers, deposits, and withdrawals with fraud detection and audit trails.

  • Instant Fund Transfers
  • Searchable History
  • Fraud Detection Layer

Secure by Design

JWT + 2FA authentication, role-based access, rate limiting, and audit middleware.

  • Two-Factor Auth (TOTP)
  • Role-Based Access Control
  • Request Audit Logging

Authentication

JWT-based authentication with Two-Factor Auth (TOTP), ASP.NET Core Identity and role-based access control.

Register

POST/api/auth/register
Public

Create a new customer account with email and password.

Login

POST/api/auth/login
Public

Authenticate and receive a JWT access token + refresh token.

Refresh Token

POST/api/auth/refresh
Public

Obtain a new access token using a valid refresh token.

Logout

POST/api/auth/logout
Auth Required

Invalidate the current session and revoke refresh token.

Enable Two-Factor Auth

POST/api/TwoFactorAuth/enable
Auth Required

Returns a QR code and secret for authenticator app setup.

Verify 2FA Code

POST/api/TwoFactorAuth/verify
Auth Required

Verify the 6-digit TOTP code from an authenticator app.

Disable Two-Factor Auth

POST/api/TwoFactorAuth/disable
Auth Required

Disable 2FA for the current user account.

Change Password

POST/api/auth/change-password
Auth Required

Change the current user's password securely.

Quick Start

Follow these steps to start using the API. All authenticated endpoints require the Authorization: Bearer <token> header.

① Register
POST /api/auth/register
{
  "email": "user@bank.com",
  "password": "SecurePass1!",
  "firstName": "John",
  "lastName": "Doe"
}
② Login
POST /api/auth/login
{
  "email": "user@bank.com",
  "password": "SecurePass1!"
}
→ returns: { "token": "eyJ..." }
③ Call APIs
GET /api/Account
Authorization: Bearer eyJ...

→ returns your accounts list
