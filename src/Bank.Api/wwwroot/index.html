QUICK START

Up and running in 3 steps

The API is live on port 5000. Register, get a token, and start calling endpoints immediately.

01

Register

Create your account with email and password.

POST /api/auth/register
{
  "email": "user@bank.com",
  "password": "SecurePass1!"
}
02

Get Token

Login and copy your JWT access token.

POST /api/auth/login
→ { "token": "eyJhbGci..." }
03

Call APIs

Add the Bearer token to every request.

GET /api/Account
Authorization: Bearer eyJhbGci...
