Secure Authentication

JWT-based auth with Two-Factor Authentication, ASP.NET Core Identity, and role-based access.

  • JWT + Refresh Tokens
  • Two-Factor Auth (2FA)
  • Admin & Customer Roles

Built with Clean Architecture

Follows Domain-Driven Design with fully separated Domain, Application, Infrastructure and API layers. Uses CQRS pattern via MediatR, AutoMapper, FluentValidation, Hangfire background jobs, and SQLite.
